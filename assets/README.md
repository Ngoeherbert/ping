# Generated Expo Assets

Binary image assets are intentionally not committed. Run the asset generator to create the local PNG files referenced by `app.json`:

```bash
npm run assets:generate
```

`npm install` also runs this generator through the `postinstall` script.

Generated files:

- `assets/icon.png`
- `assets/adaptive-icon.png`
- `assets/splash-icon.png`
- `assets/favicon.png`
- `assets/images/icon.png`
- `assets/images/splash-icon.png`
